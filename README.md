# Piano Notes

Piano Notes is an app for Piano Teachers to easily share their notes with their students.

## Navigation Links

The app includes the following navigation links:

* **Notes**: Navigate to the page that displays the notes shared by the piano teacher.
* **Students**: Navigate to the page that displays the list of students.

## Hero Element

The app includes a full-width hero element with the main title "Piano Notes" to create a strong visual impact and draw attention to the main content of the page.
